Dashboard Kas

Silakan pilih aksi untuk melihat data transaksi:

Ambil Semua Transaksi

Method: GET URL: http://localhost:8000/api/transactions

Tambah Transaksi Baru

Method: POST URL: http://localhost:8000/api/transactions

Body (JSON):{ "type": "income", "amount": 10000, "description": "Iuran anggota", "date": "2025-05-07" }

Ambil Semua Transaksi Berdasarkan Id

Method: GET URL: http://localhost:8000/api/transactions/1

Update Transaksi Berdasarkan ID

Method: PUT URL: http://localhost:8000/api/transactions/1

Body (Json) : { "type": "expense", "amount": 5000, "description": "Beli ATK", "date": "2025-05-07" }

Hapus Transaksi Berdasarkan ID

Method: DELETE URL: http://localhost:8000/api/transactions/1
